Day-10/60

Built my personal portfolio website using Claude — from resume to live site in one prompt. No templates. No drag-and-drop builders. Just AI-generated code tailored to my profile.

◆ What I Built
A modern 5-page SPA portfolio website with page-switching navigation, hero section with typing animation, skills by category, project cards with status badges, experience timeline, certifications, and contact form — all generated by Claude from my resume data.

◆ What's Inside
◆ 5-page SPA: Home → About & Skills → Projects → Experience & Certifications → Education & Contact
◆ Hero with typing animation cycling through my roles: Cybersecurity Analyst → Data Analyst → AI/ML Enthusiast → Python Developer
◆ Profile photo with gradient border + social links + download resume button
◆ Skills section with category-based cards and tech tag pills
◆ 7 project cards — 4 Finalizing (green) + 3 Upcoming (orange) — each clickable with a detail modal
◆ Experience timeline with 3 positions (Acmegrade, NIIT Foundation, NIIT Cyber Ops)
◆ 5 Course Certificates (Cisco, IBM, Acmegrade, NIIT Foundation, NIIT)
◆ Education cards with CGPA: MCA (9.52) + BCA (9.04)
◆ Dark/Light mode toggle (dark default)
◆ Fully responsive — works on mobile, tablet, desktop
◆ Glassmorphism cards, smooth page transitions, reveal animations
◆ Page indicator dots on desktop, hamburger nav on mobile
◆ All in a single HTML file — no build step, no frameworks

◆ How I Did It
◆ Uploaded my resume and profile photo to Claude
◆ Used a structured prompt with personal info, skills, projects, and design preferences
◆ Claude extracted everything from my resume and generated a complete portfolio
◆ Iteratively refined: updated experience, added Cyber Ops, corrected CGPAs, adjusted status badges
◆ One prompt → multiple refinements → one polished website

◆ Biggest Insight
Your portfolio is your first impression. Claude didn't just copy my resume — it designed a premium website around it. The lesson: AI doesn't replace your personal brand, it amplifies it. Give it your story, and it gives you a stage.
